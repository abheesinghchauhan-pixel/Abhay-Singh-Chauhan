##  Project Overview

This repository documents my work as part of the **CSE0001 – Digital Literacy** course at VIT Bhopal University. The project is structured around a scenario where I act as a **Student Digital Ambassador**, helping peers navigate the digital world responsibly and effectively.

The portfolio covers five modules:
- Visual communication through infographics
- Building a professional digital presence
- Exploring coding and collaboration platforms
- Professional email writing and etiquette
- Cybercrime awareness and online safety
